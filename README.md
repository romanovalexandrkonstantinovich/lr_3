# Проект "MobiWave" — Веб-платформа мобильного оператора

## Описание проекта

**MobiWave** — это веб-приложение, разработанное в рамках лабораторной работы №3 по дисциплине «Технологии программирования». Проект представляет собой полноценную платформу для управления услугами мобильного оператора: выбор тарифов, подключение дополнительных услуг, пополнение счёта, просмотр истории платежей, управление профилем и отзывы пользователей.

Приложение преобразовано из консольной версии (лабораторная работа №2) в WSGI-приложение с использованием микрофреймворка **Flask**. Весь ввод и вывод данных осуществляется через браузер посредством HTTP-запросов.

## Цель работы

Преобразовать консольное приложение картотеки в WSGI-приложение с использованием Flask, обеспечив переход от консольного ввода-вывода к взаимодействию через браузер.

## Функциональные возможности

### Публичные страницы
- Главная страница с отзывами пользователей
- Просмотр тарифов (Старт, Оптиум, Максимум)
- Просмотр дополнительных услуг
- Форма регистрации и входа в личный кабинет

### Личный кабинет (требуется авторизация)
- **Мой тариф** — просмотр текущего тарифа и его характеристик, возможность смены тарифа
- **Услуги** — подключение/отключение дополнительных опций (голосовая почта, HD-видео, безлимит на ночь, роуминг и др.)
- **Профиль** — редактирование персональных данных (имя, фамилия, телефон, email) и смена пароля
- **Пополнение счёта** — создание платежа с автоматическим подтверждением, просмотр истории платежей
- **Отзывы** — добавление отзывов и оценок (доступно без авторизации, но имя может быть взято из сессии)

### Административные / фоновые функции
- Автоматическое подтверждение платежей через 2 секунды после создания (в фоновом потоке)
- Отправка email-уведомления об успешном пополнении счёта (поддержка SMTP)
- Заполнение базы данных начальными отзывами при первом запуске

## Технологии

- **Backend:** Python 3.x, Flask
- **База данных:** SQLite (через модуль `db.py`)
- **Шаблонизация:** Jinja2 (HTML-шаблоны в папке `templates/`)
- **Статические файлы:** CSS, JS, изображения (папка `static/`)
- **Email:** SMTP (Gmail)
- **Дополнительно:** threading (асинхронная обработка платежей), uuid (генерация ID платежей)

## Установка и запуск

### Требования
- Python 3.8 или выше
- Установленный пакет Flask
